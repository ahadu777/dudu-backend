## 🎯 Business Context

### 用户旅程
```
用户进入小程序
  → 浏览活动/套餐
  → 微信一键登录
  → 选择日期和票数
  → 填写联系信息
  → 微信支付
  → 查看订单
  → 出示二维码
  → 商家扫码核销
```
